PROJECT TITLE : BatchGrayscale Image Converter and Iversion of Images
PURPOSE OF PROJECT : For converting and inverting Images
VERSION : v1.0.0 
DATE : 21/09/2001
HOW TO START THIS PROJECT : Use BlueJ version 4.1.1-duke, provided by https://www.dukelearntoprogram.com//course2/index.php
AUTHORS : Akshunn Sharma

USER INSTRUCTIONS :
  Download the BlueJ from the above link and install.
  Download the BatchGrayscale&ImageInversion folder as a zip from Github.
  In BlueJ, select Open Project option from File menu and select the BatchGrayscale&ImageInversion folder as the project folder.
  Some sample Images are provided in the project folder itself.
